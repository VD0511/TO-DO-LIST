# 📝 Smart To-Do List App 🚀

A sleek, responsive, and functional To-Do List application built to manage daily tasks efficiently. This app uses browser memory to ensure your tasks are never lost, even after a page refresh!

## ✨ Key Features
- **Add Tasks:** Quickly add new items to your list.
- **Persistent Storage:** Uses `localStorage` to save your data in the browser.
- **Mark as Completed:** Double-click any task to toggle a "line-through" effect.
- **Specific Deletion:** Each task comes with its own 'X' button for individual removal.
- **Clear All:** A dedicated button to wipe the entire list at once.
- **Theme Toggling:** Includes Dark Mode and Light Mode for a better user experience.
- **Fully Responsive:** Optimized for both Desktop and Mobile screens using CSS Media Queries.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Custom styling, Flexbox, and Responsive Design.
- **JavaScript (ES6):** Dynamic DOM manipulation, Arrays, and JSON handling.

## 🚀 How to Use
1. Type your task in the input field.
2. Click the **Add** button to save it to your list.
3. **Double-click** a task to mark it as finished (it will dim and show a strike-through).
4. Click the **'X'** button next to a task to delete it.
5. Use the **Dark/Light** buttons to change the app's theme.
6. Use **Clear All** to reset your entire list.

## 📱 Mobile Preview
The app is designed with a "Mobile-First" approach using `@media` queries, ensuring it looks great on iPhones, Androids, and Tablets.
